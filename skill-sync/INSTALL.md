# Installing Skills Locally

Install skills from the skill-library repo using the `npx skills` CLI. It handles platform-specific format conversion for each AI tool.

```bash
npx skills add j-alicia-long/skill-library
```

The CLI will prompt to select which skills and which AI tools (Claude Code, Cursor, Copilot, Codex, etc.) to install for.

To update after a sync:

```bash
npx skills add j-alicia-long/skill-library
```
